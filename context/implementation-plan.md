# 구현 계획 템플릿

이 문서는 `Step 00`에서 Copilot이 채워 넣을 구현 계획의 기본 자리입니다.

## 목표

`output/` 폴더에 최종 HTML 발표자료를 완성하기 위한 단계별 구현 계획을 정리한다.

## 포함 항목

1. 생성하거나 수정할 파일 목록
2. 각 파일의 역할
3. 단계별 구현 순서
4. 세팅 단계와 완성 단계의 구분
5. 배포 전 점검 항목

## 주의사항

- `Step 00`부터 `Step 03`까지는 기능 구현 코드를 직접 작성하지 않는다.
- `Step 04`부터 `output/` 폴더를 본격적으로 수정한다.
- GitHub Pages 배포를 고려한 상대 경로를 유지한다.
